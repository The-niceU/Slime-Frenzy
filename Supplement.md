# 史莱姆狂潮：最后的防线

一款基于 Python + Pygame 开发的塔防游戏，挑战无尽的怪物浪潮，通过策略性地放置和升级防御塔来保护你的阵地。

---

## 目录结构

```
史莱姆狂潮：最后的防线/
├── main.py                 # 游戏主循环与核心逻辑
├── mainMenu.py             # 菜单界面与设置
├── tower.py                # 塔与投射物类定义
├── monsters.py             # 怪物类与行为逻辑
├── waves.py                # 波数生成与难度配置
├── assets/                 # 游戏资源文件夹
│   ├── images/             # 所有游戏图片（塔、怪物、UI 等）
│   ├── audio/              # 背景音乐与游戏音效
│   └── fonts/              # 游戏字体文件
├── data/                   # 数据与文档文件夹
│   ├── saves/              # 用户游戏存档
│   └── docs/               # 游戏说明与规则文档
├── requirements.txt        # Python 依赖列表
└── README.md               # 本文件
```

## 快速开始

### 环境要求

- **Python 3.8+**
- **Pygame 2.5.2+**

### 安装步骤

#### 方式一：使用全局 Python（已安装 pygame）

直接运行：

```bash
python main.py
```

#### 方式二：使用虚拟环境（推荐）

```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
# Windows:
.\.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 运行游戏
python main.py
```

## 游戏说明

### 核心玩法

**目标**：生存尽可能多的波次，防止怪物到达终点。

**资源管理**：

- 💰 **现金**：通过击杀怪物或升级金钱塔获得，用于购买和升级防御塔
- ❤️ **生命值**：初始 100,000，被怪物到达终点时扣除，降至 0 时游戏结束

**游戏难度**：

- 🟢 **简单（Easy）**：生命值+100，防御塔成本-20%
- 🟡 **中等（Medium）**：标准难度
- 🔴 **困难（Hard）**：生命值-100，防御塔成本+20%

### 防御塔

游戏包含 **9 种防御塔**，每种有独特的攻击方式与升级路线：

| 塔名                    | 成本 | 功能                 | 特点             |
| ----------------------- | ---- | -------------------- | ---------------- |
| **飞镖塔** (Red)        | 100  | 发射飞镖攻击         | 基础塔，速度快   |
| **忍者塔** (Blue)       | 200  | 投掷飞镖，可检测隐身 | 侦查能力强       |
| **火焰塔** (Yellow)     | 300  | 喷射火焰，可点燃敌人 | 持续伤害         |
| **冰塔** (Cyan)         | 400  | 冰冻敌人，减速       | 控制能力         |
| **爆炸工厂** (Grey)     | 500  | 随机放置爆炸         | 群伤，可混乱敌人 |
| **金钱塔** (Green)      | 600  | 产出金钱箱           | 经济收益         |
| **回旋塔** (Purple)     | 700  | 发射回旋镖           | 范围伤害         |
| **超级塔** (Light Blue) | 1500 | 强力综合攻击         | 终极防御         |
| **炮塔** (Black)        | 300  | 发射炮弹             | 单体强击         |

### 怪物类型

| 怪物     | 血量 | 速度 | 特殊能力                      |
| -------- | ---- | ---- | ----------------------------- |
| 红色     | 1    | 基础 | 无                            |
| 蓝色     | 1    | 快速 | 无                            |
| 绿色     | 1    | 更快 | 无                            |
| 黄色     | 1    | 很快 | 无                            |
| 粉色     | 1    | 极快 | 无                            |
| 黑色     | 1    | 快速 | 死亡时分裂成 2 个粉色怪物     |
| 铅色     | 1    | 基础 | 仅能被火焰塔或升级冰塔击杀    |
| 紫色     | 1    | 快速 | 死亡时分裂成 4 个绿色怪物     |
| 绿色精英 | 3    | 基础 | 血量更厚                      |
| **BOSS** | 10   | 缓慢 | 死亡时分裂成 4 个绿色精英怪物 |

### 游戏控制

| 操作           | 功能                             |
| -------------- | -------------------------------- |
| **左键点击**   | 选择塔、放置塔、激活技能         |
| **Shift/Ctrl** | 长按购买模式，快速放置多个相同塔 |
| **空格**       | 切换游戏速度（1x / 2x / 暂停）   |
| **鼠标右键**   | 删除已放置的塔（需升级）         |

### 升级系统

每个塔都有 **两条升级路线**，选择不同的升级路线会获得不同的能力强化。升级消耗现金，通过左键点击升级按钮进行。

## 文件说明

### 核心模块

- **`main.py`**
  - 游戏主循环与事件处理
  - 资源加载与游戏状态管理
  - 存档读写与难度选择

- **`mainMenu.py`**
  - 主菜单、地图选择、难度选择界面
  - 存档继续功能
  - 音乐初始化

- **`tower.py`**
  - `Tower` 类：防御塔的属性、攻击、升级逻辑
  - `Projectile` 类：投射物（飞镖、炮弹等）
  - `Glaive`、`Bomb`、`Crate` 类：特殊效果实现
  - `shorten()` 函数：大数字简化显示

- **`monsters.py`**
  - `Monster` 类：怪物生成、移动、状态变化
  - 火烧、冰冻、混乱等负面效果处理
  - 分裂与死亡逻辑

- **`waves.py`**
  - `genEnemies()` 函数：根据波数与难度生成怪物组合
  - 难度曲线随波数递增

### 资源文件夹

- **`assets/images/`**
  - 游戏图片：塔、怪物、UI 按钮、背景等（PNG 格式）

- **`assets/audio/`**
  - `bgmusic.mp3`：菜单背景音乐
  - `game.mp3`：游戏进行中的背景音乐

- **`assets/fonts/`**
  - `IBMPlexSans-Regular.ttf`：游戏字体

- **`data/saves/`**
  - `Classic.txt`、`TwistyTowers.txt`：各地图存档
  - `saveFile.txt`：主存档索引

## 路径管理

项目采用**相对路径 + 基目录**的方式加载资源，确保在任意位置运行都能正确找到文件：

```python
BASE_DIR = os.path.dirname(os.path.abspath(__file__))
ASSETS_DIR = os.path.join(BASE_DIR, "assets")
DATA_DIR = os.path.join(BASE_DIR, "data")
```

这样即使移动项目文件夹，资源路径也能自动适配。

## 扩展与自定义

### 添加新塔

1. 在 `tower.py` 中增加塔的参数
2. 在 `main.py` 中添加 UI 界面
3. 在 `assets/images/` 中添加塔的图片

### 添加新怪物

1. 在 `monsters.py` 中扩展怪物类型定义
2. 在 `waves.py` 中配置怪物生成规则
3. 在 `assets/images/` 中添加怪物贴图

### 自定义地图

1. 在 `main.py` 的 `setup_Board()` 函数中添加新地图路径定义
2. 为地图创建专属贴图与背景
